# Andrea Danielle Melo Portfolio

## Project Description

This project is a responsive single-page portfolio website designed to showcase my education, leadership experience, professional background, featured projects, technical skills, and certifications.

The website was created using HTML and CSS and follows responsive web design principles.

---

## Intended User

This portfolio is designed for:

- Recruiters
- Employers
- Professors
- Entertainment industry professionals
- Marketing professionals
- Potential collaborators

---

## User Goal

The goal is for visitors to quickly learn about my professional experience, leadership roles, featured projects, technical skills, and how to contact me.

---

## Inspiration Interfaces

I used portfolio websites and editorial landing pages as inspiration.

From these websites I learned:

- Large hero sections create a strong first impression.
- Card layouts organize information clearly.
- Soft color palettes and elegant typography improve readability.
- White space makes content feel more professional.

---

## Design Choices

### Layout

A single-page scrolling portfolio with clear navigation.

### Color Palette

Cream, blush pink, olive green, and warm neutrals.

### Typography

Cormorant Garamond for headings and DM Sans for body text.

### Visual Hierarchy

The hero section is the main focal point, followed by experience, projects, skills, and contact information.

### Feedback States

Navigation links, buttons, and cards change appearance when hovered.

### Responsive Design

The hero section and all card layouts stack vertically on smaller screens.
